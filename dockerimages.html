JavaBuilder-Docker

jacknorthrup/oracle8 contains Oracle Java8
This docker image compiles and executes java . The container works just like an installed Java. However it does not effect your native install. The git repository shows how to use a Compile and Run command, to create an run java8 apps. 

https://github.com/JupyterJones/JavaBuilder-Docker
https://hub.docker.com/r/jacknorthrup/oracle8/

docker pull jacknorthrup/oracle8

Compile: docker run --rm -v $PWD:/app -w /app jacknorthrup/oracle8:latest javac
Execute: docker run --rm -v $PWD:/app -w /app jacknorthrup/oracle8:latest java
